## Definition 
---
`if` statements allow you to execute a block of code only if a specified condition is true.
## Syntax
---
```bash
if [ condition ]
then
    # code to execute if condition is true
fi
```
## Example
---
```bash
#!/bin/bash

num=10

if [ $num -gt 5 ]; then
  echo "The number is greater than 5."
fi
```
