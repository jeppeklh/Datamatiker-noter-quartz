## Definition 
---
`for` loops are used to iterate over a list of values.
## Syntax
---
```bash
for variable in list
do
  # code to execute
done
```
## Example
---
```bash
#!/bin/bash

for i in 1 2 3 4 5
do
  echo "Number: $i"
done
```
