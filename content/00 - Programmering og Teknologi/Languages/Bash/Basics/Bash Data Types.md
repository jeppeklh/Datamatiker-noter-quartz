## Definition 
---
Bash primarily handles text and strings but supports other types implicitly.

## Types of Data:
---
- **Strings**: `text="Hello"`
- **Integers**: `number=123`
- **Arrays**: `array=(1 2 3 4)`

## Example
---
```bash
#!/bin/bash

text="Hello"
number=123
array=(1 2 3 4)

echo $text
echo $number
echo ${array[0]}
```
