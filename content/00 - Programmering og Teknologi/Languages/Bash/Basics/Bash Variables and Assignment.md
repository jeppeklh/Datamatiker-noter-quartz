## Definition 
---
**Variables** in Bash are used to store data that can be accessed and manipulated throughout the script.

## Assigning Variables
---
```bash
#!/bin/bash

name="John Doe"
```
## Accessing Variables
---
```bash
#!/bin/bash

$variable_name
${variable_name}
```
## Example
---

```bash
#!/bin/bash

name="John Doe"
echo "Hello, $name"
```
